The OWASP Seeker is a most innovate open-sourced tool for detecting secrets in the code or any other text volume.
It works based on 
 - scan by regexp
 - Shanon entropy detection
 - Markov Chains detection

Using lexors allows the Seeker to parse programming languages as programming languages and improve the effectiveness of code analysis.

The latest version always can we downloaded from [Gtihub](https://github.com/bakalor/seeker).
